phpStudy 2018教程（三）

发布于 2020-10-14 | 分类于 技术分享 | 2分钟 | 287字数 | 浏览量：

phpStudy 2018 项目💾
根据PHP课堂方法，笔记而成✏️


1）

打开phpStudy

2）

首先接入上次课程输入：
show databases;
use cms;
show tables;
select * from c_user;


3）

查找c_user表内数据
输入select * from c_user where id=1;找到ID为1的数据
输入select * from c_user where name='ZYM';找到名字是ZYM的数据
输入 select * from c_user where name='ZYM' and id=1;找到名字是ZYM，ID=1的数据


4）

之后输入
alter table c_post add column p_id int primary key auto_increment;


5)

在phpStudy 2018其他选项菜单打开文件根目录

创建mysql_conn.php文件输入

    #解决乱码
    header('Content-type:text/html;charset=UTF8');
    #链接数据库
    $conn = mysql_connect('localhost','root','root');
    if(!$conn){
    	die('your database start fail').mysql_error($conn);
    }else{
    	print_r('你的数据库链接成功').'<br>';
    }
    #使用cms数据库
    $database_select = mysql_select_db('cms');

    if (!$database_select){
    	die('fail').mysql_error($database_select);
    }else{
    	print_r('你的cms数据库使用成功').'<br>';
    }
?>

最后浏览器输入地址 http://localhost/cms/mysql_conn.php显示 连接成功 就成功了

###遇到没法结束的命令行就ctrl+c
mysql -u root -p
再用上面的代码，重新输入root就好。
